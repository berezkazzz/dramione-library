# Dramione Library — версия 1.0

Сайт загружает фанфики и оценки из опубликованной Google Таблицы.  
Все посетители видят одинаковую библиотеку.

## 1. Создай Google Таблицу

1. Загрузи файл `Шаблон_Dramione_Library.xlsx` на Google Диск.
2. Открой его через Google Таблицы.
3. Заполняй лист **«Фанфики»**: одна строка — один фанфик.
4. Не меняй названия и порядок столбцов.

## 2. Опубликуй данные

В Google Таблицах:

1. **Файл → Поделиться → Опубликовать в интернете**.
2. Выбери лист **«Фанфики»**.
3. Выбери формат **CSV**.
4. Нажми **Опубликовать** и скопируй ссылку.

Важно: публикуется содержимое листа для чтения. Не добавляй личные данные.

## 3. Подключи таблицу к сайту

Открой файл `config.js` и вставь ссылку:

```js
window.DRAMIONE_CONFIG = {
  googleSheetCsvUrl: "СЮДА_ССЫЛКУ_НА_CSV"
};
```

Сохрани файл.

## 4. Обложки

### Вариант A — хранить в GitHub

1. Загрузи картинку в `assets/covers/`.
2. В Google Таблице укажи:
   `assets/covers/manacled.jpg`

Используй латинские буквы в названиях файлов.

### Вариант B — прямая ссылка

В столбец «Обложка» вставь прямую ссылку на изображение.

## 5. Загрузка на GitHub

В репозитории:

1. **Add file → Upload files**.
2. Загрузи содержимое папки `dramione-library-v1` — именно файлы, а не внешнюю папку.
3. Нажми **Commit changes**.
4. Открой **Settings → Pages**.
5. Source: **Deploy from a branch**.
6. Branch: **main**, folder: **/(root)**.
7. Нажми **Save**.

Через несколько минут появится ссылка на сайт.

## Как добавлять фанфики

Добавляй новые строки в Google Таблицу.  
После сохранения они появятся на сайте после обновления страницы. Иногда публикация Google обновляется с небольшой задержкой.

## Ограничение версии 1.0

Таблица — только источник твоих оценок. Посетители сайта не могут сохранять собственные оценки. Для аккаунтов и общего рейтинга понадобится база данных, например Supabase.
